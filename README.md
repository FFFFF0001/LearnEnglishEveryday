# 天天学英语计划

### 一.计划

1.每天学习30分钟以上的英语，前期以考研英语为基础，背单词，学语法；后期看时文，找cs科研类文献看之类。

2.所有的笔记和文献开源。可以另开Repo，但必须进到这个Repo。

### 二.背景

1.练习自己的定力，每天坚持做一件事。

2.学习自己薄弱项英语。

3.为考研做准备。

4.入门GitHub。
